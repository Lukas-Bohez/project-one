# DOS Games Folder

Place your DOS game `.zip` files in this directory.

## Where to Download DOS Games

### Official/Legal Sources:
1. **DOOM Shareware** (Free & Legal):
   - https://archive.org/download/DoomsharewareEpisode/doom.ZIP
   - File: `doom.zip`

2. **Digger** (Freeware):
   - https://archive.org/download/msdos_Digger_1983/Digger_1983.zip
   - File: `digger.zip`

3. **Commander Keen 1** (Shareware):
   - https://archive.org/download/msdos_Commander_Keen_1_-_Marooned_on_Mars_1990/Commander_Keen_1_-_Marooned_on_Mars_1990.zip
   - File: `keen1.zip`

4. **Duke Nukem 1** (Shareware):
   - https://archive.org/download/msdos_Duke_Nukem_1991/Duke_Nukem_1991.zip
   - File: `duke1.zip`

### Additional Free DOS Games:
- **Prince of Persia** (1989): https://archive.org/download/msdos_Prince_of_Persia_1990
- **Wolfenstein 3D** (Shareware): https://archive.org/download/Wolfenstein3d
- **Alley Cat** (Freeware): https://archive.org/download/msdos_Alley_Cat_1984

## Installation Instructions:
1. Download the `.zip` files from the links above
2. Place them in this `/dos-games/` folder
3. Rename files to match the expected names:
   - DOOM → `doom.zip`
   - Digger → `digger.zip`
   - Commander Keen → `keen1.zip`
   - Duke Nukem → `duke1.zip`

## File Structure:
```
dos-games/
├── doom.zip
├── digger.zip
├── keen1.zip
└── duke1.zip
```

The game will automatically load these files when you click on arcade games.
