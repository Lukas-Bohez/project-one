## ⚠️ Troubleshooting:

**If games don't load:**
- Make sure files are named EXACTLY as shown (all lowercase)
- Files must be in `/frontend/idleGame/dos-games/` folder
- Check browser console (F12) for specific error messages
- Files should be `.zip` format (not `.ZIP` or extracted folders)

**Game controls:**
- Arrow keys = Move
- Ctrl = Fire/Action
- Alt = Jump (some games)
