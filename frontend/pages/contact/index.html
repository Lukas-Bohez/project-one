Before You Contact Us

Quick Solutions - Try These First:

  • Connection Issues: Try refreshing the page or checking your internet connection
  • Quiz Not Starting: Make sure you've entered a valid session code or created a new session
  • Display Problems: Try toggling between light and dark theme, or testing in a different browser
  • Missing Content: Clear your browser cache and reload the page
  • Questions About Features: Check out our About page to learn more about what Quiz The Spire offers
