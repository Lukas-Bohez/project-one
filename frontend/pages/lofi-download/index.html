What is the Lofi Extension?

The Lofi Extension is a browser extension that brings relaxing lo-fi music and ambient sounds directly to your browser. Perfect for studying, working, or just relaxing while browsing.

✨ Key Features

  • Lo-fi Music Integration: Access curated lo-fi playlists without leaving your browser
  • Ambient Sounds: Choose from various relaxing ambient sounds
  • Easy Controls: Simple interface to play, pause, and control volume
  • Lightweight: Minimal resource usage for smooth browsing
  • Privacy-Focused: No data collection or tracking

📦 How to Install

Follow these simple steps to install the Lofi Extension in your browser:

Installation Steps

For Chromium-based Browsers (Chrome, Edge, Brave, Opera, Vivaldi, Arc, etc.):

  1. Download: Click the "Chromium Browsers" button below to download LofiExtension.zip
  2. Extract: Unzip the file to a location on your computer
  3. Open Extensions: Navigate to chrome://extensions/ (works for all Chromium browsers)
  4. Enable Developer Mode: Toggle the "Developer mode" switch in the top right corner
  5. Load Extension: Click "Load unpacked" and select the extracted folder
  6. Done! The extension icon will appear in your toolbar

For Firefox:

The Lofi Extension is now available on the official Firefox Add-ons store for easy installation:

  1. Install: Click the "Install for Firefox" button at the top of this page
  2. Follow Prompts: Follow the prompts to add the extension to Firefox
  3. Done! The extension icon will appear in your toolbar

Note: For Chromium-based browsers, manual installation is required as shown above. Firefox users can install from the official store. Keep the extracted folder for Chromium browser installations; don't delete it after installation.

🔧 Troubleshooting

Common Issues

  • Extension doesn't load: Make sure you've extracted the ZIP file completely and selected the manifest.json file (for manual installation)
  • Music doesn't play: Check your browser's audio permissions and ensure your volume is not muted
  • Can't find Developer Mode (Chromium browsers): Look in the top-right corner of the extensions page
  • Wrong version downloaded: Chromium browsers use LofiExtension.zip, Firefox uses LofiExtension-Firefox.zip
