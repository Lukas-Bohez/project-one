- Gathers feedback on UI, cross-platform support, and performance edge cases

## Repo Contributions

- Main repo manager for ConvertTheSpire and QuizTheSpire

## How to contribute

- Fork the repo, create a feature branch, and open a PR
- Keep changes small, focused, and well-documented
- All PRs are validated against lint, formatting, and production build checks via GitHub Actions
