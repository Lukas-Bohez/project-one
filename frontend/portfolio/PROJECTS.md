# Projects and Architecture

## Convert The Spire Reborn (ConvertTheSpireFlutter)

- Cross-platform Flutter app (Linux, Windows, macOS, Android) for private playlist media downloading and conversion
- Uses `youtube_explode_dart` to fetch metadata and streams, with support for 1,800+ sites, 4K/8K, and 27+ output formats
- Built for high-throughput batch conversions, local processing only, and privacy-first operation with no telemetry
- Includes embedded browser mode, ad-blocking workflow, and optional community compute donation

## QuizTheSpire

- Open-source GNU GPL 3.0 web service for playlist-to-MP3/MP4 conversion, including YouTube and 1,800+ site support
- Backend in Python (FastAPI) with Apache2 hosting, high-throughput batch conversion and privacy-first operation
- No telemetry, no accounts, no tracking; designed for privacy-preserving media workflows

## Vault The Spire

- Open-source GNU GPL 3.0 cross-platform Flutter app for private file sharing and encrypted peer-to-peer messaging
- Local encrypted vault (AES-256-GCM + SQLCipher), BitTorrent/DHT + aria2 file sharing, and X25519 ratchet E2EE chat
- QR-code contact onboarding, drag/drop torrents, magnet generation, and fully local zero-server operation
