Why does Windows show a warning?

Windows SmartScreen warns about unsigned software - this is normal for indie apps that aren’t code-signed (signing costs ~$200/year). The app is 100% open-source under the GPLv3 license.

To run the app: click “More info” → “Run anyway.”
