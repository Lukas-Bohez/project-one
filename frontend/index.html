Setup instructions

Windows Installation

  1. 1 Download ConversionTheSpireReborn.zip
  2. 2 Unzip the downloaded file
  3. 3 Open the unzipped folder and run ConvertTheSpireReborn.exe

Windows SmartScreen warning: Because the app is not code-signed, Windows may show a “Windows protected your PC” warning. This is normal for unsigned software.

  1. Click “More info”
  2. Click “Run anyway”

The app is fully open-source — you can inspect every line of code on GitHub.

Android Installation

  1. 1 Download ConvertTheSpireReborn.apk to your Android device
  2. 2 Open the downloaded .apk file
  3. 3 If prompted, enable “Install from unknown sources” for your browser
  4. 4 Tap “Install” and wait for it to finish
  5. 5 Open the app from your app drawer

Unofficial APK: This APK is not distributed through the Google Play Store. Android will warn you about installing apps from unknown sources — this is standard for any APK installed outside the Play Store.

To enable installation:

  1. Go to Settings → Security (or Settings → Apps → Special access)
  2. Enable “Install unknown apps” for your browser
  3. Try opening the APK again

The app is fully open-source — you can inspect every line of code on GitHub.

Linux Installation

  1. 1 Download linux.zip to your Linux machine
  2. 2 Unzip the file
  3. 3 Run convert_the_spire_reborn (double-click or ./convert_the_spire_reborn)

Bandwidth note: Downloads are throttled to 2 MB/s to keep the server stable for everyone.
