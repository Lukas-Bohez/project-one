What is the Lofi Extension?

The Lofi Extension is a browser extension that brings relaxing lo-fi music and ambient sounds directly to your browser. Perfect for studying, working, or just relaxing while browsing.

✨ Key Features

  • Lo-fi Music Integration: Access curated lo-fi playlists without leaving your browser

  • Lightweight: Minimal resource usage for smooth browsing
  • Privacy-Focused: No data collection or tracking

📦 How to Install

Follow these simple steps to install the Lofi Extension in your browser:

Installation Steps

For Chrome / Edge:

  1. Download: Click the "Chrome / Edge" button below to download LofiExtension.zip
  2. Extract: Unzip the file to a location on your computer
  3. Open Extensions: Navigate to chrome://extensions/ or edge://extensions/
  4. Enable Developer Mode: Toggle the "Developer mode" switch in the top right corner
  5. Load Extension: Click "Load unpacked" and select the extracted folder
  6. Done! The extension icon will appear in your toolbar

For Firefox:

⚠️ Important: Firefox requires extensions to be signed by Mozilla. For personal use, you can load it as a temporary add-on.

  1. Download: Click the "Firefox" button below to download LofiExtension-Firefox.zip
  2. Extract: Unzip the file to a permanent location on your computer (don't delete this folder!)
  3. Open Debugging: Type about:debugging#/runtime/this-firefox in Firefox's address bar and press Enter
  4. Load Temporary Add-on: Click the "Load Temporary Add-on..." button
  5. Select File: Navigate to the extracted folder and select the manifest.json file
  6. Extension Loaded: The Lofi Player icon will appear in your toolbar - click it to open the player
  7. ⚠️ Note: Temporary add-ons are removed when Firefox closes. You'll need to reload it each time you restart Firefox using the same steps.

Tip: For permanent installation in Firefox Developer Edition or Nightly, you can disable signature verification in about:config by setting xpinstall.signatures.required to false. This is not recommended for regular browsing.

Note: This is a manual installation. The extension is not yet available on browser stores. Keep the extracted folder; don't delete it after installation.

🔧 Troubleshooting

Common Issues

  • "Extension not verified" (Firefox): This is expected! Firefox requires all permanent extensions to be signed by Mozilla. Load it as a temporary add-on instead (see installation steps above).
  • "Add-on could not be installed" (Firefox): Make sure you're using "Load Temporary Add-on" from about:debugging, not trying to install the .zip directly.
  • Extension disappears after restart (Firefox): Temporary add-ons in Firefox need to be reloaded after each browser restart. This is normal behavior for unsigned extensions. Keep the extracted folder and reload using the same steps.
  • Extension doesn't load: Make sure you've extracted the ZIP file completely and selected the manifest.json file
  • Music doesn't play: Check your browser's audio permissions and ensure your volume is not muted
  • Can't find Developer Mode (Chrome/Edge): Look in the top-right corner of the extensions page
  • Wrong version downloaded: Chrome/Edge use LofiExtension.zip, Firefox uses LofiExtension-Firefox.zip
