What is the Lofi Extension?

The Lofi Extension is a browser extension that brings relaxing lo-fi music and ambient sounds directly to your browser. Perfect for studying, working, or just relaxing while browsing.

✨ Key Features

  • Lo-fi Music Integration: Access curated lo-fi playlists without leaving your browser
  • Ambient Sounds: Choose from various relaxing ambient sounds
  • Easy Controls: Simple interface to play, pause, and control volume
  • Lightweight: Minimal resource usage for smooth browsing
  • Privacy-Focused: No data collection or tracking

📦 How to Install

Follow these simple steps to install the Lofi Extension in your browser:

Installation Steps

  1. Download the Extension: Click the download button at the bottom of this page to get the LofiExtension.zip file
  2. Extract the ZIP file: Unzip the downloaded file to a location on your computer
  3. Open Browser Extensions:
    • Chrome/Edge: Navigate to chrome://extensions/ or edge://extensions/
    • Firefox: Navigate to about:debugging#/runtime/this-firefox
  4. Enable Developer Mode: Toggle the "Developer mode" switch in the top right corner (Chrome/Edge)
  5. Load the Extension:
    • Chrome/Edge: Click "Load unpacked" and select the extracted folder
    • Firefox: Click "Load Temporary Add-on" and select any file from the extracted folder
  6. Start Using: The extension icon will appear in your browser toolbar. Click it to start enjoying lo-fi music!

Note: This is a manual installation. The extension is not yet available on browser stores. Keep the extracted folder; don't delete it after installation.

🔧 Troubleshooting

Common Issues

  • Extension doesn't load: Make sure you've extracted the ZIP file completely and selected the correct folder
  • Music doesn't play: Check your browser's audio permissions and ensure your volume is not muted
  • Extension disappears after restart (Firefox): Temporary add-ons in Firefox need to be reloaded after browser restart
  • Can't find Developer Mode: Look in the top-right corner of the extensions page
