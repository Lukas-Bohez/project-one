Installation

  1. Click the download button above
  2. Unzip the downloaded file
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" (top right)
  5. Click "Load unpacked" and select the unzipped folder
  6. The extension is now installed!

Updates

When updates are available, return here to download the latest version. To update:

  1. Download the new version
  2. Unzip to a new folder
  3. In chrome://extensions/, click "Remove" on the old extension
  4. Click "Load unpacked" with the new folder
Note: This is a local extension. For automatic updates, consider publishing to the Chrome Web Store.
