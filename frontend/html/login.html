Welcome to Quiz The Spire

How to Play

Join a quiz session either locally using a SNES controller or online via this website. Answer questions correctly to earn points and items!

Items

  • Freeze: Lowers temperature by 15°C, slowing timer
  • Fireball: Raises temperature by 15°C, speeding timer
  • Answer Remove: Removes one wrong answer

Sensor Effects

The game adapts based on your environment:

  • Higher temperature = faster timer
  • Lower light = easier questions
  • More noise = easier questions
