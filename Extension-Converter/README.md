# YouTube Video Converter Extension

A Chrome extension to convert YouTube videos to MP3/MP4 with variable quality.

## Features
- Convert YouTube videos to MP3 or MP4
- Variable quality options (default: best)
- Uses cookies from the YouTube tab for authentication
- Standalone extension, no backend required

## Installation
1. Download the extension files.
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extension folder.

## Usage
1. Navigate to a YouTube video.
2. Click the extension icon.
3. Select format (MP4/MP3) and quality.
4. Click "Convert & Download".

## Limitations
- Playlists are not supported yet.
- Requires internet connection for conversion service.

## Privacy
This extension sends YouTube URLs and cookies to a third-party conversion service (y2mate.com) for processing. No data is stored by the extension.
