# Convert the Spire - Firefox Installation Guide

## ⚠️ Firefox Exclusive Extension

This extension **only works on Firefox** due to Chrome's Manifest V3 restrictions that prevent downloading YouTube content.

## Quick Start

1. **Download** ConvertTheSpire.zip from [quizthespire.com](https://quizthespire.com/frontend/html/converter.html)
2. **Extract** the ZIP to a permanent location (don't delete this folder!)
3. **Open Firefox** and type `about:debugging#/runtime/this-firefox` in the address bar
4. **Click** "Load Temporary Add-on..."
5. **Select** the `manifest.json` file from the extracted folder
6. **Visit YouTube** and start converting!

## Important Notes

- ✅ This is a **temporary add-on** in Firefox
- 🔄 You'll need to **reload it after each Firefox restart**
- 📁 **Keep the extracted folder** - don't move or delete it
- 🦊 **Firefox 142+** required

## Features

- 🎵 Convert YouTube videos to MP3 or MP4
- 📋 Batch convert entire playlists automatically
- ⚙️ Choose quality and format options
- 📊 Real-time download progress tracking
- 🔒 Privacy-focused (all processing happens locally)

## How to Use

1. Navigate to any YouTube video or playlist
2. Click the "Convert the Spire" icon in your toolbar
3. Select format (MP3/MP4) and quality
4. Click "Convert & Download"
5. Watch progress in the extension popup

## Troubleshooting

### Extension Not Loading
- Make sure you selected `manifest.json` (not the folder)
- Check that Firefox is version 142 or higher
- Verify the extracted folder is complete

### Downloads Not Starting
- Check Firefox's download permissions
- Ensure you have write access to the download folder
- Try selecting a different download location

### Extension Disappeared After Restart
- This is normal for temporary add-ons
- Simply reload using steps 3-5 above
- The extension keeps your settings

## Why Firefox Only?

Chrome's Manifest V3 introduced severe restrictions on:
- Downloading content from YouTube
- Accessing certain web APIs needed for conversion
- Background processing capabilities

Firefox's Manifest V2 still supports these features, making it the only browser where this extension can function properly.
